Embedded Projects

RISC-V Interpreter

RISC-V instruction set simulator developed in Python to help beginners understand the workings of a 5-stage single-cycle processor. The interpreter correctly executes a subset of RISC-V instructions including ADD, SUB, LW, SW, BEQ, JAL, while simulating registers, memory, and program flow. By providing detailed execution metrics such as instruction count, cycle count, and CPI, this tool enables learners to visualize processor behavior, experiment with RISC-V programs, and gain hands-on experience with low-level instruction execution, memory management, and control flow in a simplified yet accurate environment.

Key Skills: Python | RISC-V | Visual Studio Code.
